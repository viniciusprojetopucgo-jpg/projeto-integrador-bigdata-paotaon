# 🥖 Projeto Integrador IV-A: Inteligência de Dados Pão Tá On

## 📋 Descrição
Este projeto faz parte do Projeto Integrador da **PUC-GO** (Big Data e IA). O objetivo é realizar a análise estratégica de vendas de uma operação de delivery, utilizando processamento distribuído para gerar insights sobre faturamento por bairro e performance de canais.

## 🛠️ Stack Tecnológica
* **Processamento:** PySpark (Apache Spark)
* **Ambiente Cloud:** Databricks Community Edition
* **Banco de Dados:** PostgreSQL (hospedado via Railway)
* **Linguagem:** Python

## 🏗️ Arquitetura da Solução
A solução consome dados via conexão **JDBC**, realiza transformações no Spark e gera visualizações interativas.

## 📊 KPIs Analisados
1. **Faturamento por Bairro:** Identificação das regiões com maior volume financeiro.
2. **Performance de Canais:** Comparação entre iFood e Site Próprio.
3. **Volumetria de Itens:** Ranking de produtos para otimização de estoque.

## 🚀 Como visualizar
* O arquivo `.ipynb` pode ser visualizado diretamente pelo GitHub.
* O arquivo `.html` contém a versão interativa do dashboard para visualização em navegadores.
